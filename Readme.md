# 北京理工大学邮箱推送服务

### 功能
1. 可添加`关键词`进行`监测`，发现相关邮件将`推送`到设定平台
2. 邮箱信息`汇总`推送到设定平台，目前有`自建推送平台`设置、PushPlus[^1]推送设置
3. 关键词监测`时间间隔`与信息汇总`推送时间`均可`自定义`
4. 监测时间范围自定义：`今日`、`7日内`等
